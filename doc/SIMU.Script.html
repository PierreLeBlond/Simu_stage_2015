Class: Script

SIMU.Script

new Script()

Represent a script used to load data from files
Properties:
Name Type Description
name String name of the script
script function the script itself /!\ The script will be given the result attribute of File object. The script MUST return an array of object as followed : [{name:"index", value:}, {name:"position", value:}, {name:"color", value:}, {name:"additional data", value:}, ...] where "index" & "position" are mandatory, the rest is optional
binary boolean True if the file should be read with the readAsArrayBuffer function, else readAsBinaryString will be used
