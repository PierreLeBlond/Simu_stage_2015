Class: Snapshot

SIMU.Snapshot

new Snapshot()

Represent some data snapshot, i.e. fixed data at a certain time
Properties:
Name Type Description
isReady boolean True if the buffers have been correctly populated with data, therefor they aren't null
position Float32Array Position buffer
direction Float32Array Direction buffer
directionIsSet boolean True if there is another Snapshot ready, which is the next snapshot in time. The direction between the two snapshot is then ensure to be set and computed
index Float32Array Index buffer
color Float32Array Color buffer
{@link SIMU.Octree} octree - Octree set from the position & the index buffer
info Array Array of buffer, storing different information about the point
